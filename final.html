1) What is Full Stack Development?

Full Stack Development involves building both the front-end and back-end of a web application. A Full Stack Developer can design the interface, manage server logic, and interact with databases.

  • Front-End: HTML, CSS, JavaScript for UI design.
  • Back-End: Server-side code, authentication, and data management.
  • Integration: Connecting client requests with database responses.
User → Browser → Request → Server → Database
← Response ← Data ← Server

2) HTML

HTML (HyperText Markup Language) provides the structure of a web page. It defines headings, paragraphs, images, and links using semantic tags.

Example:
<article>
  <h3>Classic T-Shirt</h3>
  <p>Price: $20</p>
</article>

This markup represents a product card with a heading and price.

3) CSS

CSS (Cascading Style Sheets) styles HTML elements. It controls color, layout, and responsiveness for better user experience.

Example:
article {
  background: #f4f4f4;
  padding: 15px;
  border-radius: 10px;
}

This CSS snippet creates a visually appealing product card.

4) Bootstrap Framework

Bootstrap is a front-end framework with pre-styled components and a responsive grid system. It speeds up development and ensures consistency across devices.

Example:
<div class="card" style="width: 18rem;">
  <img src="tshirt.jpg" class="card-img-top" alt="T-shirt">
  <div class="card-body">
    <h5 class="card-title">Classic T-Shirt</h5>
    <p class="card-text">$20</p>
    <a href="#" class="btn btn-primary">Buy Now</a>
  </div>
</div>

Bootstrap’s grid and components simplify responsive design.

5) JavaScript

JavaScript makes websites interactive. It allows event handling, form validation, and dynamic updates without reloading the page.

Example (Cart Demo):

Cart: Empty

<button onclick="addToCart('T-shirt')">Add to Cart</button>

<script>
let cart = [];
function addToCart(item) {
  cart.push(item);
  document.getElementById("cartList").innerText = cart.join(", ");
}
</script>
